# Мой конфиг для neovim
Просто конфиг для vim опубликованный тут для быстрого доступа с любой машины.

## Setup

Клонируйте в каталог с конифгурацией neovim, в *nix-ах по умолчанию это
~/.config/nvim:
<pre>
git clone https://github.com/tre3k/nvim-conf.git ~/.config/nvim
</pre>

Запустите nvim (первый запуск будет с ошибками, т.к. в конфиге конфигурация
плагинов, которые еще не установлены. Наверное в nvim & lua есть механизмы
проверки установленных плагинов, но мне лень в этом разбираться, "на скорость
не влияет").
При первом запуске скачается и установится vim-plug.
И запустите установку плагинов:
<pre>
:PlugInstall
</pre>

## Cтруктура каталога
<pre>
├── init.lua                    -- точка входа
├── lua
│   ├── colorscheme.lua         -- оформление цветовой схемы / темы
│   ├── hotkeys.lua             -- бинды клавиш
│   ├── options.lua             -- опции редактора (номера строк, отступы и.т.д)
│   └── plugins.lua             -- подключаемые плагины
└── README.md
</pre>
